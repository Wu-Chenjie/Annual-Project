## 本轮优化项

1. 将仿真重构为配置驱动，减少硬编码参数。
2. 采用预分配数组替代循环内 append，降低内存分配开销。
3. 通过领航机平滑机动参数与航点切换半径优化，显著降低编队误差。
4. 增加主机加速度低通滤波，减小前馈抖动。
5. 增加批量评测脚本，支持多随机种子复现实验与 JSON 导出。

## 优化后结果

主入口 `main.py` 实测（默认配置）：

- 运行耗时约 `2.3s`
- 三架从机平均误差分别约 `0.1805m`、`0.2110m`、`0.1999m`
- 全部小于 `0.3m` 阈值

批量评测 `simulations/benchmark.py`（5 组随机种子）：

- 平均运行时间约 `2.591s`
- 从机平均误差均值约 `0.2027m`、`0.2022m`、`0.1991m`
- `0.3m` 阈值全通过
